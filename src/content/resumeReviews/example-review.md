# Software Engineer Resume Review

In this review, we analyze a software engineer's resume targeting international
positions.

## What Works Well

- Clear formatting and structure
- Quantified achievements
- Relevant technical skills highlighted

## Areas for Improvement

- Add more context to project descriptions
- Include international-friendly certifications
- Consider adding a brief summary section

## Recommended Changes

1. Move skills section higher
2. Add metrics to each role
3. Include relevant keywords from job descriptions
